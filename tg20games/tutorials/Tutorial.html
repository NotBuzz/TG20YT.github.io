Tutorial: How to install the Friday Night Funkin’ Week 7 Chrome OS App

After reading this tutorial, you will know how to install the Friday Night Funkin’ Week 7 app for your Chrome OS device. Note that this was tested on a Chromebook running Chrome OS version 100.0.4896.82 which was released on April 8, 2022, and these steps may be different in future versions of Chrome OS, therefore making this outdated if that ever happens.

Step 1: Download the Friday Night Funkin’ Week 7 zip file from the TG20 Games website or itch.io page.

Step 2: Open the zip file and copy the crx from it into a folder of your choice.

Step 3:

Head over to chrome://extensions in a new tab and enable developer mode. It’s located in the top right of the page.

Step 4: Go into the folder you just extracted the crx from in step 2 and drag&drop it into the extensions page. A prompt will appear after that asking if you want to install the app “Friday Night Funkin’ Week 7.” Click add app and it will install.

(sorry for crappy colors, had to compress it so that it would upload)

Step 5: Open the app launcher and look for the newly installed application, and open it and let it start.

And BOOM! You’ve finished this tutorial! Hope you found this helpful. Enjoy!
